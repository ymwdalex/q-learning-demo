# Q\_learning_demo

## Dependencies
```
Python 2.7
tkinter
```

## Human play
``` sh
python human_player.py --difficulty 0|1|2
```

## AI play
``` sh
python ai_player.py --difficulty 0|1|2 --max-iteration 20
```
